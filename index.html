Scalearn


Simple library for statistics, time series analysis, and machine learning in Scala

Download .zip Download .tar.gz View on GitHub

Welcome to Scalearn

This library would help you to deal with a bunch of data by making use of statistics, time series analysis, and machine learning. Although such academic domains seem to be difficult for most developers ( including me! ) , the main target users of this library are ordinary scala developers; in that sense, it focus on usability rather than speed or efficiency. Besides, I hope this could be useful for beginners of scala.

Prerequisite

  • scala
  • sbt
  • Homebrew (optional)

Installation

  1. Download zip file of this library

  2. Start your terminal ( if you are using mac ), and go to root directory

    3. $ cd root_directory
  3. If you don't have sbt, install through homebrew 

    4. $ brew install sbt
  4. At root directory, type "sbt package"( assume you need to install sbt first. Homebrew makes it easier to install both scala and sbt)

    5. $ sbt package
  5. Finally, you got jar file at root/target/scala-2.X/here

Quick Start

If you are interested in statistics, you need to import "datafactory package" and "Converter object" first of all: they allows you to obtain statistical value easily. 

import datafactory._
import Converter._
